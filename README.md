# HideToggler

Contains a JQuery plugin for quickly setting a link for toggling a div

## Installation

Add this line to your application's Gemfile:

    gem 'hide_toggler'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install hide_toggler
